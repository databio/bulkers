# bulkers

Multi-container environment manager.

Port from python to rust. 

Alpha testing.

## Install

```bash
curl -sL https://github.com/databio/bulkers/releases/latest/download/bulkers-Linux-x86_64.tar.gz | tar xz && sudo mv bulkers /usr/local/bin/
```

**macOS (Apple Silicon):**
```bash
curl -sL https://github.com/databio/bulkers/releases/latest/download/bulkers-macOS-aarch64.tar.gz | tar xz && sudo mv bulkers /usr/local/bin/
```

**macOS (Intel):**
```bash
curl -sL https://github.com/databio/bulkers/releases/latest/download/bulkers-macOS-x86_64.tar.gz | tar xz && sudo mv bulkers /usr/local/bin/
```

Or build from source: `cargo install --path .`
